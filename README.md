# Copy All Tabs URLs

A simple Chrome extension to copy URLs of all open tabs.

**Copy All Tabs URLs** is an intuitive and efficient Chrome extension designed to help you instantly copy the URLs of all open tabs in your browser window. With just a single click, the extension gathers the URLs and copies them to your clipboard, ready for you to paste wherever you need.

![Copy All Tabs URLs](screenshots/copy-all-tabs-urls.png)

Whether you're working on research, sharing resources with friends, or keeping track of your favorite websites, **Copy All Tabs URLs** streamlines the process for you.

## Key Features

- **One-click URL copying**  
  Copy all open tab URLs instantly with a single click using a simple, user-friendly interface.

- **Automatic URL formatting**  
  URLs are automatically formatted as a clean list, with each URL on a new line for easy reading, organizing, and sharing.

- **Real-time notifications**  
  Receive instant notifications confirming that the URLs have been successfully copied to your clipboard.

- **Lightweight and efficient**  
  Designed to be fast and lightweight, ensuring zero impact on your browsing performance.

- **Privacy-focused**  
  Your data stays yours. Copy All Tabs URLs does not collect or store any personal information or browsing data.
